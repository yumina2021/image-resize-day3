# 画像サイズ変換器

シンプルなブラウザ上で動く画像リサイズ／圧縮ツールです。生成されるファイルはダウンロードされます。

## 使い方

1. `index.html` をブラウザで開く（ファイルを直接開くか、ローカルサーバーを使ってください）。
2. 「画像を選択」で画像を選ぶ。
3. 幅・高さ・品質・出力形式を設定。
4. 「変換してダウンロード」を押すと変換後の画像がダウンロードされます。

## ローカルでの簡単な起動（推奨）

Pythonがある場合:

```bash
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## 備考

- 完全に静的なクライアント実装なので、そのままGitHubにアップロードしてGitHub Pagesで公開できます。
- PNGでは品質指定が無視される場合があります（ブラウザ依存）。

作業が必要なら、GitHub用に `.gitignore` や `LICENSE` を追加します。
